Library Table Concepts

Three design approaches for the library view table interface

Concept A: Classic Sortable Table

A clean, traditional table inspired by iTunes/Spotify list views. Sortable columns, alternating row backgrounds, and hover highlights. Good balance of data density and readability.

Concept B: Data-Dense Compact View

A spreadsheet-inspired dense view for power users. Smaller row height, minimal decoration, monospace numbers for alignment. Maximizes visible data. Inspired by Airtable/Notion database views.

Design Notes

  • Concept A is best for users who want a familiar spreadsheet-like experience with clear sorting
  • Concept B maximizes data density for power users managing large libraries (500+ books)
  • Concept C prioritizes visual appeal and is easier to scan for users who value aesthetics
  • All concepts use the Warm Library theme tokens (Crimson Pro font, sienna accents, cream backgrounds)
  • Click column headers in A/B to see sort indicator behavior (demo only)
  • Mobile: A/B would need horizontal scroll; C could stack vertically
